# whkd Shortcuts

## Prefix

| Key | Meaning |
| --- | --- |
| `Win` | `lwin` |

## System

| Shortcut | Action |
| --- | --- |
| `Win+Esc` | Pause/resume whkd |
| `Win+Shift+Esc` | Toggle Komorebi shortcut helper |
| `Win+Ctrl+Shift+O` | Reload Komorebi configuration |

## Apps

| Shortcut | Action |
| --- | --- |
| `Win+Ctrl+Enter` | Open Windows Terminal |

## Window

| Shortcut | Action |
| --- | --- |
| `Win+Ctrl+Q` | Close focused window |
| `Win+Ctrl+M` | Minimize focused window |
| `Win+Ctrl+T` | Toggle floating |
| `Win+Ctrl+F` | Toggle monocle |

## Focus

| Shortcut | Action |
| --- | --- |
| `Win+Ctrl+H` | Focus left |
| `Win+Ctrl+J` | Focus down |
| `Win+Ctrl+K` | Focus up |
| `Win+Ctrl+L` | Focus right |
