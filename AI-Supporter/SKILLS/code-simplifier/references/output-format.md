# Output Format

Use this report structure after a simplify run.

## Findings

- `reuse`: concise findings or `No reuse issues found.`
- `quality`: concise findings or `No quality issues found.`
- `efficiency`: concise findings or `No efficiency issues found.`

## Applied Fixes

- list only the fixes that were actually made
- if no edits were applied, say `No fixes applied.`

## Validation

- list the checks you ran and their result
- if validation could not run, say that explicitly

## Deferred

- list findings you intentionally did not fix because they were broad, risky, or outside scope
- if none, say `No deferred findings.`
