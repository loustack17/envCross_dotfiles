# Shared Skills

This directory is the source of truth for shared skills.

Installed links:
- `~/.claude/skills`
- `~/.codex/skills`
- `~/.config/opencode/skills`

Current external skills vendored into this tree:
- `apify-actor-development`
- `apify-generate-output-schema`
- `apify-ultimate-scraper`

Local shared skills in this tree can be used without relying on Claude plugin cache paths.
